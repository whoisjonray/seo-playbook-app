Full Strategy Details

Use Richly Descriptive Alt Text on All Images

What It Is: Provide detailed, descriptive alt attributes for all images on the site.
Why It Works: Alt text helps Google understand image content for image search indexing and improves accessibility, which indirectly supports SEO and user experience.
When To Use: All websites, especially publishers, ecommerce, and multimedia-heavy sites.
Execution Steps:

  1. For each image, write an alt description that accurately describes what the image shows or its context.
  2. Avoid keyword stuffing; keep descriptions natural and helpful.
  3. Include images in both post content and UI elements where appropriate.
    Pitfalls & Limits:
  • Thin, generic alt text or keyword stuffing can hurt SEO.
  • Missing alt text loses an opportunity to rank in Google Images.
    Example: New York Post’s use of rich alt text for all images is a notable best practice that contributes to their SEO strength.
